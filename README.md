## Run Locally

Clone the project

```bash
  git clone https://github.com/faruuhan/mymovie-app-react.git
```

Go to the project directory

```bash
  cd mymovie-app-react
```

Create .env file

```bash
  REACT_APP_API=PUT_YOUR_API_FROM_TMDB
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
